# 忘年会 クロスBINGO

忘年会で使えるインタラクティブなクロスBINGOゲームです。

## 機能

- 3x3のBINGOカード
- お題がランダムに配置される（中央マスは固定）
- 達成した人の会社名と名前を入力できる
- 達成したマスはオレンジ色に変わる

## お題の編集方法

中央マス以外のお題は `tasks.js` ファイルで管理されています。

1. `tasks.js` ファイルを開く
2. `BINGO_TASKS` 配列内にお題を追加・編集
3. ファイルを保存してGitHubにプッシュ

### 例

```javascript
const BINGO_TASKS = [
    'ゴルフが趣味',
    'サウナが好き',
    '勤続20年以上',
    '御社（ホスト企業）と一番付き合いが長い',
    '出身地が九州'
];
```

**注意：**
- 各お題は文字列として `'` または `"` で囲んでください
- お題の間はカンマ（`,`）で区切ってください
- **改行位置は自動判定されます**（HTML側で長いテキストは自動的に改行されます）
- 括弧や句読点がある場合は、その位置で優先的に改行されます
- 中央マスは「アダストリアブランドのアイテムを身に着けている」で固定されています
- 3x3のBINGOカードなので、中央以外に8マス分のお題が必要です
- お題が8個より多い場合は、ランダムに8個が選ばれます
