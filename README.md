# 忘年会 クロスBINGO

忘年会で使えるインタラクティブなクロスBINGOゲームです。

## 機能

- 3x3のBINGOカード
- お題がランダムに配置される（中央マスは固定）
- 達成した人の会社名と名前を入力できる
- 達成したマスはオレンジ色に変わる

## お題の編集方法

中央マス以外のお題は `tasks.csv` ファイルで管理されています。

1. `tasks.csv` ファイルを開く
2. お題を記入し、お題とお題の間は空行で区切る
3. お題内で改行したい場合は、実際に改行を入れる（HTML側で自動的に`<br>`タグに変換されます）
4. ファイルを保存してGitHubにプッシュ

### 例

```
ゴルフが趣味

サウナが好き

勤続20年以上

御社（ホスト企業）と
一番付き合いが長い

出身地が
九州
```

**注意：**
- お題とお題の間は必ず空行で区切ってください
- お題内で改行したい場合は、そのまま改行を入れるだけでOKです（`<br>`タグは不要）
- 中央マスは「アダストリアブランドのアイテムを身に着けている」で固定されています
- 3x3のBINGOカードなので、中央以外に8マス分のお題が必要です
- お題が8個より多い場合は、ランダムに8個が選ばれます
